<h1 align="center">⚡ AI Compliance Intelligence Platform</h1>

<p align="center">
An AI-powered system that transforms regulatory documents into trackable compliance tasks and monitors deadlines through an intelligent dashboard.
</p>

<p align="center">
🚀 FastAPI • ⚛ React • 🗄 MySQL • 🤖 AI/NLP
</p>

<hr>

<h2>📌 Project Overview</h2>

<p>
Mining operations are governed by strict regulatory frameworks to ensure <b>environmental protection</b>, 
<b>worker safety</b>, and <b>operational compliance</b>. Managing these regulations manually can be complex and error-prone.
</p>

<p>
The <b>AI Compliance Intelligence Platform</b> automates compliance monitoring by extracting regulatory obligations 
from documents and converting them into structured compliance tasks. The platform provides a centralized dashboard 
to track deadlines, detect risks, generate alerts, and create compliance reports.
</p>

<hr>

<h2>🚀 Key Features</h2>

<ul>
<li>📄 <b>AI Regulation Parsing</b> – Extract compliance rules automatically from regulatory PDFs</li>
<li>📋 <b>Automated Task Creation</b> – Convert regulations into structured compliance tasks</li>
<li>📊 <b>Compliance Monitoring Dashboard</b> – Visualize project compliance status and deadlines</li>
<li>⚠ <b>Smart Alerts System</b> – Identify upcoming or overdue compliance requirements</li>
<li>📈 <b>Risk Prediction</b> – Analyze compliance risk using project task statistics</li>
<li>🔍 <b>Bottleneck Detection</b> – Detect delays in compliance workflows</li>
<li>📑 <b>Automated Report Generation</b> – Generate compliance reports instantly</li>
<li>📉 <b>Data Visualization</b> – Interactive charts for risk distribution</li>
</ul>

<h2>⚙ Installation Guide</h2>

<h3>Backend Setup</h3>

<pre>
cd backend
pip install -r requirements.txt
uvicorn main:app --reload
</pre>

<h3>Frontend Setup</h3>

<pre>
cd frontend
npm install
npm run dev
</pre>
